# Aloha Surf

A small static site built to practice **HTML** and **CSS**.

**Aloha Surf** is a fictional surfboard rental company. This project is for learning only, not a real business.

## Contents

- `index.html` — page structure and content
- `styles.css` — layout and styling
- `images/` — assets used on the page

🌐 Live site: [rgs7.github.io/aloha-surf](https://rgs7.github.io/aloha-surf/)
